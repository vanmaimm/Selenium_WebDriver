# Selenium_WebDriver
## Webdriver
 ```
 Chrome: chromedriver
 Firefox: geckodriver
 Edge: EdgeDriver
 ```
 ## How to setup
 ```
 1. Download & use Eclipse or IntelliJ IDEA
 2. Download webriver (chromedriver, geckodriver, EdgeDriver)
 - https://chromedriver.chromium.org/downloads
 - https://github.com/mozilla/geckodriver/releases
 - https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/
 Note: choose webdrive version suitable with browser version 
 
 ```
